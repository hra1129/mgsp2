--------------------------------------------------------------------------------
8Kbyte bank

Bank	Desc.
#00		[Boot code]
#01		[MGSDRV]		(※) 先頭13byte は動作には必要ない部分なので、13byte は #00側に はみ出して 置く。
							 最後に、動作に必要ない部分があり、これが #01 をはみ出す。
#02		[MGSP本体]
#03〜	[KanjiROM]
#xx〜	[曲データ]

--------------------------------------------------------------------------------
Boot code
	基本的な情報収集
		自身のスロット番号調査
		page1 の RAM slot検索
		page2 の RAM slot検索
		C000h番地以降に置くワークの初期化
		MGSDRVを page1 の RAM の 6000h-7fffh へ転送
		MGSDRVを初期化
		グラフィックパーツを初期化
		MGSP本体を page1 の RAM の 4000h-5fffh へ転送

--------------------------------------------------------------------------------
KanjiROMデータのすぐ次ぎから、曲データリスト。
曲データリストは、ソートする関係から、DRAM にコピーして使われる。MGSP本体を置くRAMの
後半。
