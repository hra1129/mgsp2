1. はじめに
	MegaROMカートリッジ用のソフト作成の練習として作りました。
	ROM上の曲しか再生できませんし、どんなに大きなROMを用意しても
	最大192曲までしか対応していません。
	一方で、通常版 MGSP v2.1.1 と異なり、DOS2/Nextor や 64KBを越える
	RAMは必要ありませんし、MSX1用の メモリマッパー非対応の拡張RAM
	カートリッジも利用可能です。
	SCCも、SCC搭載ゲームカートリッジに対応していますので、
	準備しなければならない機材の敷居が下がっています。
	MSXをBGM再生マシンとして使いたい場合には、このROM版の方が使いやすい
	かもしれません。

2. 使い方 (ROMイメージの作成方法)
	ROMイメージ生成ツールは、Windows10用です。

	(1) MGSフォルダの中に、登録したい MGSファイルをすべてコピーします。
	    少なくとも１つは入れなければなりません。

	(2) BUILD.BAT をダブルクリックして下さい。2つのROMファイルが生成されます。
		MGSPASC8.ROM .... メガコンが ASCII8Kタイプ のカートリッジに書き込む場合用
		MGSPKNM8.ROM .... メガコンが Konami8Kタイプ のカートリッジ(SCC/SCC-Iでも可)に書き込む場合用

	MGSフォルダ内には mgslist.txt が出来上がります。１行１ファイルで、
	この行の記述順によってentry順の順番が決まります。
	mgslist.txt を書き替えてから BUILD.BAT を実行すると、その順に変化します。

3. カスタマイズ
	SRC/BASE_CUSTOM.ASM の中身を書き替えてカスタマイズして下さい。
	カスタマイズ後は、BUILD.BAT を実行することで ROMファイルも更新されます。
